# intel-standards module

Provides canonical VirtuaSOC intelligence tradecraft primitives:

- Source reliability scale (A–F) with labels and descriptions.
- Confidence levels (High/Moderate/Low) with descriptive metadata.
- Risk evaluation helpers to build a full 5×5 matrix with derived risk categories.
- Action item helper to normalize owner/deadline/description fields.

The module is dependency-free and designed for re-use by product generators, renderers, and delivery adapters.
