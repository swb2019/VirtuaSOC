# intel-standards

Domain primitives that encode intelligence community grading systems used across VirtuaSOC products.

## Contents

- Source reliability scale (A-F) with human-readable definitions.
- Confidence level descriptors (High/Moderate/Low).
- Helpers to calculate and enumerate 5x5 risk matrices and severity bands.
- Action item helper that enforces owner/deadline metadata for task tracking.

All exports are pure TypeScript utilities without external dependencies.
