# intel-standards

Canonical intelligence-analysis primitives shared by VirtuaSOC products.

## Contents

- **Source reliability**: AF-style A-F scale with descriptors.
- **Analytic confidence**: High / Moderate / Low helpers.
- **Risk scoring**: 5x5 likelihood/impact matrix with configurable thresholds
  and derived ratings.
- **Action items**: Typed helpers that normalize owners, titles, and deadlines.

These utilities are deterministic, pure, and safe to serialize for audits.
