# intel-standards

Canonical primitives for VirtuaSOC intelligence products:

- Source Reliability (AF scale A-F) with descriptors.
- Confidence levels (High/Moderate/Low) plus ordering helper.
- 5x5 risk matrix + derived `RiskScore` utility.
- Action item primitive enforcing owner + ISO deadline.

All exports are pure utilities designed for reuse across the generator,
renderer, and delivery modules.
