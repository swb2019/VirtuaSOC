# intel-standards

Primitive building blocks for VirtuaSOC intelligence products:

- NATO-style source reliability codes (A–F) with helper descriptions.
- Confidence levels (High/Moderate/Low).
- Deterministic 5x5 risk matrix + risk scoring helpers.
- Action item helper enforcing owner + deadline semantics.

All exports are pure TypeScript utilities designed for reuse by product generators,
renderers, and delivery adapters.
