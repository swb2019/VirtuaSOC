# intel-standards Module Docs

Defines reusable primitives that align VirtuaSOC outputs with ICD-203 and
ISO-31000 requirements:

- AF source reliability scale (A-F) descriptors.
- Analytic confidence levels (High/Moderate/Low).
- A 5×5 likelihood/impact risk matrix with deterministic scoring and ratings.
- Action item helper enforcing owner + ISO deadline.

See `SPEC.md` for acceptance criteria and `CONTRACT.md` for the frozen interface.
