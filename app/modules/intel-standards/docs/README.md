# intel-standards module

Provides canonical primitives for intelligence production:

- Source reliability scale (A–F) with helper metadata and validation.
- Confidence levels (High/Moderate/Low) plus comparison utilities.
- 5x5 risk matrix helpers with derived risk score + rating bands.
- Minimal action-item representation with ISO deadline normalization.

All exports live in `app/modules/intel-standards/src/index.ts` and are covered by Vitest unit tests.
